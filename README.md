🌾 AgriTrack – Field Inspection & Crop Advisory Platform
AgriTrack is a web-based Agriculture Assistant Management System that helps field officers manage farmer records, conduct field inspections, and provide crop advisory services digitally. The system replaces manual paperwork with an organized, centralized platform for better efficiency and transparency.

🚀 Features
-> Secure Login & Authentication
-> Farmer Registration & Management
-> Field Inspection Report Creation
-> Crop Health Monitoring
-> Pest & Disease Observation Tracking
-> Dashboard with Summary Statistics
-> Search, Update & Delete Records


🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap
Backend: Java, JSP, Servlets
Database: MySQL
Server: Apache Tomcat
